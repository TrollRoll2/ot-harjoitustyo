The program is meant to be a game that the user is able to play.

There will only be one type of user, at least to begin with.

The program will upon startup provide the user with a main menu. In the main menu the user can press a button with their mouse in order to start the game, but they can also exit the game or change some options.

I have planned to create some sort of platformer game that allows the user to move a character around, collect items and complete levels.
